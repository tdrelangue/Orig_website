Ce que je fais What I do

  • Audit d'accessibilité Accessibility audits

    Tests WCAG 2.2 AA (clavier + lecteurs d'écran). Rapport priorisé. WCAG 2.2 AA testing (keyboard + screen readers). Prioritized report.

  • Audit d'utilisabilité cognitive Cognitive usability audits

    Charge cognitive, architecture de l'information, compréhension des parcours. Cognitive load, information architecture, and journey comprehension.

  • Audit de sobriété numérique Digital sobriety audits

    Poids des pages, performance, bonnes pratiques d'éco-conception (RGESN). Page weight, performance, and eco-design best practices.

  1. Cadrage Scoping Périmètre, objectifs, contraintes. Scope, objectives, constraints.
  2. Audit Audit Tests clavier/lecteurs d'écran + revue UI. Keyboard/screen reader tests + UI review.

  4. Recommandations Recommendations Actions priorisées, effort/impact. Prioritized actions, effort/impact.
